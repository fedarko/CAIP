How to use
----------

-Download and install Pygame if you don't have it yet.
See http://www.pygame.org/ for details. (Pygame is licensed
under the LGPL version 2.1, and is thus compatible with the
GPL version 3; see http://www.gnu.org/licenses/gpl-faq.html#AllCompatibility)

-Run run.py to start the program.

-You can modify level1 in Code/Levels.py to
change the starting states of the cells
in a level.

-You can change variables in Code/Config.py to
impact the simulation in various ways.
